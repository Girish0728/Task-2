# Task-2
Stock Portfolio Tracker
This is a simple Python program developed as part of the CodeAlpha Python Programming Internship. It helps users calculate the total value of their stock investments.

Description
The Stock Portfolio Tracker allows users to enter stock names and quantities, and it calculates the total investment based on predefined stock prices.

Objective
To build a Python program that uses basic concepts like dictionaries, loops, and conditions to solve a real-world problem.
